Screening access

Include/Exclude decisions are shared with your whole team by writing them directly to the GitHub repository. To do that from your browser, you need a GitHub personal access token with write access to this repo. It's stored only in your own browser (localStorage) and sent only to api.github.com — never anywhere else.

Create one at GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Repository access: only this repo → Permissions: Contents: Read and write.
